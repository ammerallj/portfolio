Building Collaborative Workspaces

A live object people recognize on sight, wherever it lands.

Timeline
2020–2022
Role
UX Designer · Interaction Design · Platform Design · Visual Design
Team
Design · Product Management · Engineering · Research

Overview

As part of Microsoft Loop's founding design team, I defined the interaction foundations that made live collaborative components recognizable and usable across Microsoft 365.

I shaped the persistent controls for identity, sharing, and collaboration; the visual cues for presence and live editing; and the product's early visual identity, while helping ship the first Loop components into Teams.

See where Loop landed → Explore Microsoft Loop (opens in new tab)

Impact

  • Unified how every Loop component behaves across Microsoft 365 by designing the Loop control—the persistent interaction model for identity, sharing, and collaboration—later documented as the standard component structure for developers and patented as Interactive user interface controls for shared dynamic objects (US 12,277,305 (opens in new tab)).
  • Shipped Microsoft's first Loop components into Teams, enabling live collaborative tables, lists, agendas, and action items that stayed synchronized wherever they were shared—the Fluid (opens in new tab) work that later became Microsoft Loop and debuted at Ignite 2021 (opens in new tab).
  • Defined the component structure other developers now build to: Microsoft's public developer documentation (opens in new tab) specifies the header and border that mark a Loop component as live and portable.
